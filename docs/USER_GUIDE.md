# 一天 · 用户手册

> 欢迎使用「一天」—— 你的24小时生命画布

---

## 什么是「一天」？

「一天」是一个时间追踪与生活记录工具。它把你的每一天切分成24个格子，让你清晰地看到时间去了哪里、心情如何变化、目标完成得怎样。

核心理念很简单：**记录一天不难，坚持记录每一天就能看见一生的轮廓。**

---

## 快速上手

### 第一步：启动应用

```bash
cd yi-tian
npm install
npm start
```

打开浏览器访问 http://localhost:3000

### 第二步：认识界面

左侧是导航栏，包含：
- **仪表板**：今天的总览，包括时间轴、目标、习惯
- **数据统计**：全局分析图表
- **活动管理**：管理你的活动类型库
- **设置**：自定义偏好

### 第三步：记录你的第一个小时

1. 在仪表板页面，点击24小时时间轴中任意一个格子
2. 从弹出的活动选择器中选择一个活动
3. 完成！记录已保存

## 核心功能详解

### 24小时时间轴

时间轴是「一天」的核心。每个格子代表一个小时：

- **灰色格子**：尚未记录
- **彩色格子**：已记录活动（颜色代表活动分类）
- **蓝色边框格子**：当前小时
- **点击格子**：记录或更改活动

时间轴下方的彩色条显示了今天的时间分配比例。

### 活动管理

「一天」预置了8大分类、60多种活动：

| 分类 | 示例活动 |
|------|----------|
| 💼 工作 | 深度工作、会议、代码编写 |
| 🏠 生活 | 用餐、家务、通勤 |
| 💪 健康 | 运动、跑步、瑜伽、冥想 |
| 📚 学习 | 阅读、在线课程、语言学习 |
| 💬 社交 | 朋友聚会、家庭时光 |
| 🎮 娱乐 | 看剧、游戏、摄影 |
| 😴 休息 | 睡眠、放松、发呆 |
| 🌱 个人成长 | 复盘反思、目标规划 |

你也可以创建自定义活动：
1. 进入「活动管理」页面
2. 点击「新建活动」
3. 填写名称、分类、图标和颜色

### 目标管理

每天设定3-5个小目标，追踪完成情况：

1. 仪表板的「今日目标」区域会显示今天的目标
2. 点击圆圈标记为完成
3. 完成率会实时更新

### 心情追踪

记录每小时的心情和精力状态：

- 心情评分：1（很差）到 10（非常好）
- 精力水平：1（精疲力竭）到 10（精力充沛）
- 可选记录压力和专注度

### 习惯打卡

培养好习惯，戒掉坏习惯：

1. 进入设置或仪表板的习惯区域
2. 创建你想追踪的习惯
3. 每天点击打卡
4. 查看连续天数和最佳纪录

### 每日总结

一天结束时，花2分钟写下：

- 今天最亮的点
- 今天的遗憾
- 感恩什么
- 学到了什么
- 明天的计划

## 数据统计

### 仪表板统计卡片

- **已记录时间**：今天已记录的小时数
- **目标完成率**：今日目标完成百分比
- **平均心情**：今日心情平均值
- **连续记录**：连续多少天进行了记录

### 数据统计页面

- **时间分配饼图**：各类活动的时间占比
- **分类明细**：每个分类的详细数据
- **全局统计**：总天数、总条数、总时间

- 可通过设置页面导出 JSON 或 CSV 格式备份

## 常见问题

**Q：数据会丢失吗？**
A：数据存储在 `data/yi-tian.db` 文件中，只要不删除这个文件，数据就不会丢失。建议定期备份。

**Q：如何更换端口？**
A：在 `.env` 文件中设置 `PORT=你想要的端口号`。

**Q：如何重新填充示例数据？**
A：运行 `npm run seed -- --force`。

**Q：可以多人使用吗？**
A：当前版本为单用户设计。多用户支持在规划中。

**Q：如何清空所有数据？**
A：删除 `data/yi-tian.db` 文件，重启服务器即可。
